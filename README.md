# 🏦 Bank System

A simple Bank System project built using Node.js.

---

## 🚀 Getting Started

Follow these steps to run the project locally:

### 1️⃣ Clone the repository

```bash
git clone https://github.com/heba-mohamedd/Bank_System.git
```

### 2️⃣ Navigate to project folder

```bash
cd Bank_System
```

### 3️⃣ Install dependencies

```bash
npm install
```

### 4️⃣ Run the project

```bash
npm run dev
```

---

## 🛠️ Technologies Used

* Node.js
* Express
* MongoDB (if used)

---

## 📌 Notes

* Make sure you have Node.js installed on your machine.
* You can add environment variables in a `.env` file if needed.
